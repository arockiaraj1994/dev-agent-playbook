1. Add the server (one line)

Run once in your terminal — this writes to your user-level Claude Code config:

{% if auth_enabled %}
claude mcp add --transport sse {{ server_label }} {{ sse_url }} --header "Authorization: Bearer {{ token_val }}"
{% else %}
claude mcp add --transport sse {{ server_label }} {{ sse_url }}
{% endif %}

Use this single-line form. Multi-line versions with trailing \ often fail with missing required argument 'commandOrUrl' when a stray space sneaks in after the backslash.

2. Or: add via JSON (recommended when using headers)

The add-json form passes the whole config as one argument and avoids any flag-parsing surprises:

{% if auth_enabled %}
claude mcp add-json {{ server_label }} '{"type":"sse","url":"{{ sse_url }}","headers":{"Authorization":"Bearer {{ token_val }}"}}'
{% else %}
claude mcp add-json {{ server_label }} '{"type":"sse","url":"{{ sse_url }}"}'
{% endif %}

The JSON must be wrapped in single quotes so the shell doesn't interpret the inner double quotes.

3. Project-scoped install (optional)

To make the server available only inside one repository, run the same command from inside the repo with --scope project:

{% if auth_enabled %}
cd ~/code/my-app
claude mcp add-json {{ server_label }} --scope project '{"type":"sse","url":"{{ sse_url }}","headers":{"Authorization":"Bearer {{ token_val }}"}}'
{% else %}
cd ~/code/my-app
claude mcp add-json {{ server_label }} --scope project '{"type":"sse","url":"{{ sse_url }}"}'
{% endif %}

4. Verify

List MCP servers to confirm it registered:

claude mcp list

5. Remove

To remove the server from Claude Code:

claude mcp remove {{ server_label }}

For a project-scoped install, run the same command from inside the repo with --scope project.

      }

  }

{
  "mcpServers": {
    "{{ server_label }}": {
      "url": "{{ sse_url }}"
    }
  }
}
{% endif %}

2. Project-scoped install

For per-project config, create .cursor/mcp.json at your repo root with the same structure. Cursor merges project + user configs at runtime.

3. Restart Cursor

Fully quit and reopen Cursor (don't just reload the window). Open the MCP panel in settings to see the server's status indicator turn green.
